# Movix

A modern movie discovery web application that allows users to explore trending, popular, and top-rated movies and TV shows using real-time data from TMDB API.

The platform provides a responsive and interactive user experience with detailed movie information, trailers, ratings, cast details, and search functionality.

---

## Live Demo

https://movixfailmes.netlify.app/

---

## Features

- Browse Trending Movies & TV Shows
- Explore Popular and Top Rated Content
- Search Functionality
- Detailed Movie Information
- Responsive Design
- Dynamic Hero Banner
- Infinite Scrolling
- Trailer & Video Preview
- Cast and Crew Information
- Similar Movie Recommendations

---

## Technologies Used

- React.js
- JavaScript (ES6+)
- Redux Toolkit
- Axios
- React Router DOM
- SCSS / CSS
- TMDB API

---

## Functionality

The application allows users to:

1. Browse movies and TV shows by category
2. Search for movies dynamically
3. View detailed information about selected content
4. Watch trailers and related videos
5. Explore cast, ratings, genres, and release dates
6. Discover recommended and similar movies
